# 構成パターン
下に向かって、可用性は高くなるが、その分コストが高くなる

| 構成パターン           | [[RPO]]              | [[RTO]]        |
| ---------------------- | ---------------- | ---------- |
| バックアップ・リストア | 数時間以内       | 24時間以内 |
| パイロットライト       | 数分以内         | 数時間以内 |
| ウォームスタンバイ     | 数秒以内         | 数分以内   |
| マルチサイト           | なし、または数秒 | 数秒以内           |
