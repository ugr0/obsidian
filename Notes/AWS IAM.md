# リソースベースかIDベースか

| |種類|概要|
|---|---|---|
|①|IDベース|いわばIAMのこと。「誰がどのリソースに対して○○できる」|
|②|リソースベース|例えばS3のバケットポリシー。「このリソースに対して誰が○○できる」|
